# customer-churn-analysis-python-sql-powerbi
Customer churn analysis project using Python, SQL and Power BI to identify churn drivers and improve customer retention.
# Sales Performance Dashboard

## Project Overview
This project analyses sales performance using a retail Superstore dataset. The goal is to identify revenue trends, top-performing products, regional performance, customer segments, and business opportunities.

## Business Problem
Businesses need to understand what drives sales performance so they can focus on the right products, regions, and customer segments.

## Tools Used
- Python
- Pandas
- SQL
- Power BI
- Excel
- GitHub Codespaces

## Key Questions
- What is the total sales performance?
- Which products generate the highest sales?
- Which regions perform best?
- Which customer segment contributes the most revenue?
- Are there seasonal sales trends?

## Key Insights
- A small number of products generate a large share of revenue.
- The Consumer segment contributes strongly to sales.
- Some regions outperform others consistently.
- Sales show seasonal trends across time.

## Business Recommendations
- Focus marketing on top-performing products.
- Prioritise high-performing regions.
- Plan inventory around peak sales periods.
- Use customer segmentation to improve targeting.
